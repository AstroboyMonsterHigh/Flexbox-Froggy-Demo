📋 Quick Reference Card
Row (default)
Items: ← →
justify-content: horizontal
align-items: vertical
Row-Reverse
Items: → ←
justify-content: horizontal (reversed)
align-items: vertical
Column
Items: ↑ ↓
justify-content: vertical
align-items: horizontal
Column-Reverse
Items: ↓ ↑
justify-content: vertical (reversed)
align-items: horizontal
🎯 Common Use Cases
📱 Navigation Bar
flex-direction: row
Use for horizontal menus and navigation links.
📝 Sidebar Layout
flex-direction: column
Perfect for vertical navigation or stacked content.
📱 Mobile Views
@media (max-width: 768px) { flex-direction: column; }
Switch from row to column on smaller screens.
🌐 RTL Languages
flex-direction: row-reverse
Use for right-to-left languages like Arabic or Hebrew.
