## 📖 Flexbox Reference Guide
Use these properties within the sandbox to see how they transform your layout.

### 🏁 Container Properties (The Parent)
| Property | Description | Common Values |
| :--- | :--- | :--- |
| `flex-direction` | Sets the main axis | `row`, `column`, `row-reverse` |
| `justify-content` | Aligns items on the main axis | `flex-start`, `center`, `space-between` |
| `align-items` | Aligns items on the cross axis | `stretch`, `center`, `flex-end` |
| `flex-wrap` | Forces items onto multiple lines | `nowrap`, `wrap`, `wrap-reverse` |
| `align-content` | Aligns multiple lines | `space-around`, `stretch`, `center` |

### 🧩 Item Properties (The Children)
| Property | Description | Common Values |
| :--- | :--- | :--- |
| `order` | Changes the visual order | `1`, `-1`, `0` |
| `flex-grow` | Ability for an item to grow | `0`, `1`, `2` |
| `flex-shrink` | Ability for an item to shrink | `1`, `0` |
| `align-self` | Overrides parent alignment for one item | `center`, `flex-end`, `baseline` |
