# ✨ Flexbox Wonderland ✨

> **"If you can't learn the rule, build a world where the rule becomes visible."**

This project is a part of my **Digital Lab** series. It was born out of a personal challenge: I found CSS Flexbox logic difficult to visualize, so I leveraged HTML and Vanilla JavaScript to build an interactive playground that makes the concepts tangible.

## 🚀 The Mission
The goal of this "test" repository was to create a sandbox where I could break and fix Flexbox properties in real-time. By building the tool myself, I mastered the CSS properties I was previously stuck on.

## 🎮 Features
- **Individual Property Demos:** Dedicated sections for `justify-content`, `align-items`, `flex-direction`, and more.
- **Interactive "Order" Sliders:** Real-time visual reordering of items using the `order` property.
- **The Sandbox Arena:** A "Pick & Mix" zone where multiple properties can be combined to see how they interact (and sometimes conflict!).
- **Security-First UI:** Clean, responsive design built with a focus on user experience and code readability.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for the playground.
- **CSS3 (Flexbox):** The heart of the project.
- **Vanilla JavaScript:** Powers the live style manipulation without the need for external frameworks.
